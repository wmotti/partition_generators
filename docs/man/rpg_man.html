Abstract

Manual of the python module random_partition_generator

Synopsis

From a python interpreter or another script:

from random_partition_generator import RandomPartitionGenerator as RPG

rpg = RPG(n, poset, [optional_parameters])

or interactively from the command line:

python random_partition_generator.py

At the end of the computation, interesting variables are:

rpg.regular_partitions
rpg.counter_regular_partitions

and

rpg.open_partitions
rpg.counter_open_partitions

Their values can be displayed using print.

Description

Generates n random partition of the given poset with uniform probabilities.

Parameters

Parameter Description Default value
n number of partitions to be created
poset a list of relations between two elements of the set
check if true, check if the partition is regular and open true
print_stats if true, print statistics about partitions at the end true
pbar if true, show a progress bar during the computation true

Relationships in poset are between elements i and i+1, where i is even.

Example of a simple poset specification
poset = ["a", "b", "c", "d"]

describes a poset of 4 elements with relations between ("a","b") and ("c","d").
